# CSS 工具类

## 文字

| 类名 | 效果 |
| --- | --- |
| `.d2-text-center` | 文字水平居中 |

## 浮动

| 类名 | 效果 |
| --- | --- |
| `.d2-fl` | 左浮动 |
| `.d2-fr` | 右浮动 |
| `.d2-clearfix` | 清除浮动 |
